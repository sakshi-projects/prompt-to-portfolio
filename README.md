# 🎬 Chris Columbus | 3D Portfolio Website

## 📖 Overview

The **Chris Columbus 3D Portfolio Website** is an interactive, visually engaging single-page portfolio dedicated to the acclaimed filmmaker **Chris Columbus**. Designed with modern web technologies, the website showcases his career, iconic films, achievements, visual legacy, and contact information through immersive animations and 3D-inspired effects.

The project emphasizes creativity and user experience by combining smooth scrolling, animated transitions, hover effects, and responsive design to create a cinematic browsing experience.

---

## ✨ Features

### 🏠 Home Section

* Attractive hero section with a cinematic theme
* Animated title and subtitle
* Smooth scrolling navigation

### 👤 About Section

* Introduction to Chris Columbus
* Professional biography
* Career highlights
* Interactive image with 3D hover effect

### 🎥 Filmography

* Showcases iconic films directed by Chris Columbus
* Interactive 3D flip cards
